Codex로 고전 미니 게임 스페이스 인베이더를 다시 만들어 보세요.

### 아래 공식 사이트에서 Git·Python·VS Code 기본 도구 설치하기

```
# Git 공식 사이트
https://git-scm.com/

# Python 공식 사이트
https://www.python.org/

# VS Code 공식 사이트
https://code.visualstudio.com/
```

### Git으로 스페이스 인베이더 게임 코드 가져오기

```
git clone https://github.com/akDeveloper/70s-space-invaders.git
```

### Python 가상환경을 만들고 의존성 설치하기

```
# --- 클론한 70s-space-invaders 폴더 안에서 실행 ---

# 가상환경 생성
mkdir .venv && cd .venv
python3 -m venv .
cd ..

# 가상환경 활성화
source .venv/bin/activate

# 의존성 설치
pip install pygame
```

### 프로젝트 실행하기

```
python main.py
```

### Vibe coding 프롬프트

- 프로젝트에 배경음악과 효과음을 추가하세요.
- 승리 또는 패배 후 화면에 메시지를 표시하세요.
- 난이도 레벨 1~10을 추가하고 레벨이 올라갈 때마다 침략자 속도를 2배로 올리세요. 각 게임 종료 후 다음 레벨 시작, 재시작, 계속 진행 중 어떤 키를 누를지 안내하세요.
