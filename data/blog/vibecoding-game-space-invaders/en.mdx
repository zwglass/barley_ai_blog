Use Codex to recreate the classic mini game Space Invaders.

### Install the basics — Git, Python, Node.js, VS Code — from the official sites below

```
# git official site
https://git-scm.com/

# python official site
https://www.python.org/

# node.js official site
https://nodejs.org/

# vscode official site
https://code.visualstudio.com/
```

### Install the Codex CLI

```
npm install -g @openai/codex
```

### Configure Codex environment variables (add the relay settings when using a proxy inside China)

```
# Open ~/.codex/config.toml and add the following lines to the top of the file:

model_provider = "crs"
model = "gpt-5-codex"
model_reasoning_effort = "high"
disable_response_storage = true
preferred_auth_method = "apikey"
[model_providers.crs]
name = "crs"
base_url = "https://{relay-domain}/openai"
wire_api = "responses"
requires_openai_auth = true
env_key = "CRS_OAI_KEY"
```

### Enter the Codex terminal

```
export CRS_OAI_KEY={api_key}
codex
```

### Clone the Space Invaders game with Git

```
git clone https://github.com/akDeveloper/70s-space-invaders.git
```

### Create a Python virtual environment and install dependencies

```
# --- Run these commands inside the cloned 70s-space-invaders folder ---

# Create the virtual environment
mkdir .venv && cd .venv
python3 -m venv .
cd ..

# Activate the virtual environment
source .venv/bin/activate

# Install dependencies
pip install pygame
```

### Run the project

```
python main.py
```

### Vibe coding prompts

- Add background music and sound effects to the project.
- After a win or loss, display an on-screen message.
- Add difficulty levels 1–10; each level doubles the invaders’ speed and, after every game, prompt which key starts the next level, restarts, or continues play.
