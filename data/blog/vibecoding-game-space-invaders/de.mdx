Setze Codex ein, um das klassische Minispiel Space Invaders neu zu erstellen.

### Installiere die Basis-Tools – Git, Python, Node.js, VS Code – über die offiziellen Seiten unten

```
# Git-Webseite
https://git-scm.com/

# Python-Webseite
https://www.python.org/

# Node.js-Webseite
https://nodejs.org/

# VS Code-Webseite
https://code.visualstudio.com/
```

### Codex-CLI installieren

```
npm install -g @openai/codex
```

### Codex-Umgebungsvariablen konfigurieren (füge die Relay-Einstellungen hinzu, wenn du in China über einen Proxy arbeitest)

```
# Öffne ~/.codex/config.toml und füge die folgenden Zeilen am Anfang ein:

model_provider = "crs"
model = "gpt-5-codex"
model_reasoning_effort = "high"
disable_response_storage = true
preferred_auth_method = "apikey"
[model_providers.crs]
name = "crs"
base_url = "https://{relay-domain}/openai"
wire_api = "responses"
requires_openai_auth = true
env_key = "CRS_OAI_KEY"
```

### Codex-Terminal starten

```
export CRS_OAI_KEY={api_key}
codex
```

### Klone das Space-Invaders-Spiel mit Git

```
git clone https://github.com/akDeveloper/70s-space-invaders.git
```

### Erstelle eine Python-Umgebung und installiere Abhängigkeiten

```
# --- Führe diese Befehle im geklonten Ordner 70s-space-invaders aus ---

# Virtuelle Umgebung erstellen
mkdir .venv && cd .venv
python3 -m venv .
cd ..

# Virtuelle Umgebung aktivieren
source .venv/bin/activate

# Abhängigkeiten installieren
pip install pygame
```

### Projekt starten

```
python main.py
```

### Vibe-Coding-Eingaben

- Füge dem Projekt Hintergrundmusik und Soundeffekte hinzu.
- Zeige nach einem Sieg oder einer Niederlage eine Bildschirmmeldung an.
- Ergänze Schwierigkeitsgrade 1–10; jede Stufe verdoppelt die Geschwindigkeit der Invasoren und nach jedem Spiel soll eine Aufforderung anzeigen, welche Taste die nächste Stufe startet, neu startet oder fortsetzt.
