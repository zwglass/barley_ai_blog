Utilisez Codex pour recréer le mini-jeu classique Space Invaders.

### Installez les outils de base — Git, Python, Node.js, VS Code — depuis les sites officiels ci-dessous

```
# Site officiel de Git
https://git-scm.com/

# Site officiel de Python
https://www.python.org/

# Site officiel de Node.js
https://nodejs.org/

# Site officiel de VS Code
https://code.visualstudio.com/
```

### Installer la CLI Codex

```
npm install -g @openai/codex
```

### Configurer les variables d’environnement Codex (ajoutez les paramètres de relais si vous passez par un proxy en Chine)

```
# Ouvrez ~/.codex/config.toml et ajoutez les lignes suivantes en tête de fichier :

model_provider = "crs"
model = "gpt-5-codex"
model_reasoning_effort = "high"
disable_response_storage = true
preferred_auth_method = "apikey"
[model_providers.crs]
name = "crs"
base_url = "https://{domaine-relais}/openai"
wire_api = "responses"
requires_openai_auth = true
env_key = "CRS_OAI_KEY"
```

### Entrer dans le terminal Codex

```
export CRS_OAI_KEY={api_key}
codex
```

### Clonez le jeu Space Invaders avec Git

```
git clone https://github.com/akDeveloper/70s-space-invaders.git
```

### Créez un environnement virtuel Python et installez les dépendances

```
# --- Exécutez ces commandes dans le dossier cloné 70s-space-invaders ---

# Créer l’environnement virtuel
mkdir .venv && cd .venv
python3 -m venv .
cd ..

# Activer l’environnement virtuel
source .venv/bin/activate

# Installer les dépendances
pip install pygame
```

### Exécutez le projet

```
python main.py
```

### Prompts de vibe coding

- Ajoutez une musique de fond et des effets sonores au projet.
- Après une victoire ou une défaite, affichez un message à l’écran.
- Ajoutez des niveaux de difficulté 1 à 10 ; chaque niveau double la vitesse des envahisseurs et, après chaque partie, indiquez quelle touche lancer le niveau suivant, redémarrer ou continuer.
