Codex を使ってクラシックなミニゲーム「スペースインベーダー」を再現しましょう。

### 必要なツール（Git・Python・Node.js・VS Code）を下記の公式サイトからインストールする

```
# Git公式サイト
https://git-scm.com/

# Python公式サイト
https://www.python.org/

# Node.js公式サイト
https://nodejs.org/

# VS Code公式サイト
https://code.visualstudio.com/
```

### Codex CLI をインストールする

```
npm install -g @openai/codex
```

### Codex の環境変数を設定する（中国国内でリレー接続を使う場合のプロキシ設定を追加）

```
# ~/.codex/config.toml を開き、ファイルの先頭に次の設定を追加します。

model_provider = "crs"
model = "gpt-5-codex"
model_reasoning_effort = "high"
disable_response_storage = true
preferred_auth_method = "apikey"
[model_providers.crs]
name = "crs"
base_url = "https://{リレードメイン}/openai"
wire_api = "responses"
requires_openai_auth = true
env_key = "CRS_OAI_KEY"
```

### Codex ターミナルに入る

```
export CRS_OAI_KEY={api_key}
codex
```

### Git でスペースインベーダーのゲームコードを取得する

```
git clone https://github.com/akDeveloper/70s-space-invaders.git
```

### Python の仮想環境を作成し、依存関係をインストールする

```
# --- クローンした 70s-space-invaders フォルダ内で実行 ---

# 仮想環境を作成
mkdir .venv && cd .venv
python3 -m venv .
cd ..

# 仮想環境を有効化
source .venv/bin/activate

# 依存パッケージをインストール
pip install pygame
```

### プロジェクトを実行する

```
python main.py
```

### Vibe coding のプロンプト

- プロジェクトに BGM と効果音を追加する。
- 勝利または敗北後に画面へメッセージを表示する。
- 難易度レベル 1〜10 を追加し、レベルが上がるたびにインベーダーの移動速度を 2 倍にする。各ゲーム終了後、次のレベル開始・再スタート・続行のどのキーを押すかを表示する。
